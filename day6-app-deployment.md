# Day 6 - Web Application Deployment on AWS EC2

## 🚀 What I Did

- Deployed a Python-based web server on an AWS EC2 instance
- Configured inbound rules to allow traffic on port 8000
- Created and served a custom HTML page (index.html)
- Exposed the application publicly via EC2 public IP
- Monitored incoming HTTP requests through server logs

---

## 🌐 Result

Successfully hosted and accessed a web application on a cloud server using a public IP address.

---

## 📊 Observations

- HTTP requests were logged in real time on the server
- Browser automatically requested additional assets (e.g., favicon), resulting in expected 404 responses
- The server correctly handled incoming traffic from external clients

---

## 💡 Key Learning

- Practical experience in deploying and exposing applications on cloud infrastructure
- Understanding of networking basics such as ports, public IP access, and request handling
- Reinforced concepts of remote server management and application hosting
